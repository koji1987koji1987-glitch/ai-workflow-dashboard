# ai-workflow-dashboard
AI-powered engineering workflow prototype

Summary:
This system is designed to minimize the burden on managers when allocating work to engineers. 

It not only avoids allocating work to busy engineers (reducing workload) but also analyzes skills and performance to suggest the best engineer for a task.

Adopted technologies:
HTML, CSS, JavaScript, Chart.js, Microsoft Access, Git / GitHub

Characteristics:
Dashboard, It recommends a task, Performance analysis, Related database, Rule-and-condition-based AI

Repository Contents:
- index.html → Dashboard user interface
- style.css → UI styling and layout
- script.js → Task recommendation logic
- Architecture_Diagram.png → System architecture diagram
- Database2.accdb → Database schema and sample dataset
- dashboard-screenshot 1.png → Dashboard screenshot
- dashboard-screenshot 2.png → Analytics screenshot
- AI_Integration_Strategy.txt → AI decision strategy

AI logic;
The program avoids engineers with a lot of workload, and analyzes skills and their performances to conclude which developer is the best for a task.
